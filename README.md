## Installation & starting the App [Linux, Windows and MacOS]

- Install (minimum) JDK 17 as your default java
- Download or checkout the source repository
- enter the ```trifa_material``` directory
- run:
  -  Linux :```./gradlew run```
  -  Windows: ```gradlew.bat run```
  -  MacOS :```./gradlew run```
  -  raspi 64bit: ```export MESA_EXTENSION_OVERRIDE="-GL_ARB_invalidate_subdata" ; ./gradlew run```
  -  Asahi Linux: ```export MESA_EXTENSION_OVERRIDE="-GL_ARB_invalidate_subdata" ; export MESA_GL_VERSION_OVERRIDE=3.0 ; ./gradlew run```
